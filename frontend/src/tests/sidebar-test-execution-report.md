
2. **Auto-close Behavior**:

   ```typescript

      if (isMobile) {
         useUIStore.getState().setSidebarOpen(false);
      }
   }, [isMobile]);

   React.useEffect(() => {
      handleMobileClose();
   }, [location.pathname, handleMobileClose]);
   ```
